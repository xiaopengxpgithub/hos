## hos对象存储服务
可以用来存储图片,文件,视频,音频等内容

### 技术架构
* 1.java后台框架:springboot+mybatis <br/>
* 2.数据库: MySQL <br/>
* 3.大数据相关技术:Hadoop+Hbase+zookeeper <br/>
